Tentang Saya

Nama Saya Yoga Pratama, teman-teman memanggil saya Yoga. Saya lahir di Boyolali 15 Desember 1996. Saya tinggal di daerah Cileungsi, tepatnya di Perum. Duta Mekar Asri.

Hobi saya adalah membaca buku, terutama buku Komputer. Kedepannya saya akan belajar lebih keras lagi agar Cita-cita saya menjadi Web Developer menjadi kenyataan.

Saat ini saya sedang menempuh pendidikan di Politeknik LP3I Jakarta, dengan Program Studi Informatika Komputer. Kemudian, bahasa programming yang sedang saya pelajari adalah VB.NET, PHP(Codeigniter & Laravel), HTML, CSS, dll.
